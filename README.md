# Face Similarity AI (Python)

This project uses a Teachable Machine image model with TensorFlow and OpenCV
to detect which famous person a face looks like in real time using a webcam.

## Features
- Real-time webcam face prediction
- Shows the most similar celebrity with confidence percentage
- Press **Q** to stop the camera

## Requirements
- Python 3.10
- TensorFlow
- OpenCV
- NumPy

## How to Run
```bash
pip install tensorflow opencv-python numpy
python main.py
